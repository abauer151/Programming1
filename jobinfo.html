Software Engineer

Description: On the most basic level, software engineers write, debug, maintain, and test software that instructs a computer to accomplish certain tasks, such as saving information, performing calculations, etc. A software engineer will convert what needs to happen into one of many programming languages, the most common of which being Java, C++ and Python.

Average Salary: $106,000

Employers: Liberty Mutual, Lincoln Financial, and Cgi Technologies

Related Careers: Hardware Engineer, Network Engineer, and Software Developer
